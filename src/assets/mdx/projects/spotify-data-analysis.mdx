Created a data analysis and visualization project using Python libraries such as Pandas, Matplotlib, and Seaborn as part of a high school project.

The project focused on extracting, cleaning, and analyzing Spotify’s Extended Streaming History dataset (multi-file JSON data obtained via GDPR export). The pipeline handled data normalization, aggregation, and feature extraction to enable meaningful analysis of listening behavior.

### Graphs and Visualizations Created

- Popularity analysis of top artists, albums, and tracks
- Scatter plots and frequency distributions for daily and hourly listening activity
- KDE distribution plots for average track playtime and listening patterns
- Correlation heatmaps to analyze relationships between artists and listening metrics

### Interactivity

Implemented a fuzzy search pipeline using [RapidFuzz](https://rapidfuzz.com/) to enable approximate string matching on artist names. This allows users to query artists and dynamically view their associated listening statistics and analysis.
